# Globe Plotter

Visualise GeoJSON on a semi transparent globe. Made possible by [Michael McLoughlin's Globe project](https://github.com/mmcloughlin/globe)

# Demo

[On Heroku](http://globe-plotter.herokuapp.com/)

# Dependencies

## Go
- Globe
- geojson

## JavaScript
- Spectrum 
- Skeleton CSS
- Node UUID
- turf.js
- csv2geojson (Mapbox)
